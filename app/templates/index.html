ขั้นตอนใช้งาน

  1. เลือกโฟลเดอร์ USB ปลายทาง
  2. ตรวจสอบและสร้างโฟลเดอร์ OPL ที่จำเป็น
  3. นำเข้าไฟล์ ISO
  4. ระบบอ่าน Game ID จาก ISO และบันทึก mapping ใน CFG/game_manifest.json
  5. ค้นหา ART และตรวจดูภาพตัวอย่าง
  6. บันทึก ART ที่เลือกโดยใช้ Game ID เดียวกับเกม
